Vault_X 🔐

A secure password manager application written in Rust.

⚠️ Important Notice

Vault_X is still under development. Security cannot yet be fully guaranteed. Please do not use it as your primary password manager for important or sensitive data.

Features

- Secure password storage
- User-friendly interface
- Written in Rust for maximum performance

Installation and Usage

1. Clone or download the project
2. Navigate to the project directory
3. Run `cargo build`
4. Run `cargo run`
5. Follow the instructions in the application
